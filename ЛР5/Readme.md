Лабораторная 5.1 (2 балла): Разработать приложение, использую фреймворк Django. В данном приложении необходимо реализовать следующий функционал:

* Разработать несколько (более двух) моделей базы данных, которые внутри себя содержат несколько (более трёх) различных полей. Также необходимо между моделями реализовать отношения один ко многим (Foreign Key) и многие ко многим (ManyToMany).
* На странице администрирования необходимо отразить все имеющиеся модели. В каждой модели отобразить только часть её полей (если в модели три поля, отобразить два их них) и отфильтровать значения по одному из выведенных полей.

Подготовить электронный отчет (1 балл), в котором необходимо описать все используемые сторонние функции, также описать собственные переменные, функции, классы и т.д.
